## tslint
 `tslint` 설정시 코드 스타일(개발 스타일) 커스터마이징을 할 수 있다.
 
#### tslint 설치
```
$ npm install tslint
$ tslint --init // 설정파일 생성
```
### vscode 설정
#### vscode extention 설치
    1. cmd + p 
    2. **ext install tslint** 설치

### inteliJ
- `preferences` > `typescript` >  `tslint`로 설정
