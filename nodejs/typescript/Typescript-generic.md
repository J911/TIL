# 타입스크립트 제네릭

## Generic이란
- 클래스나, 함수의 정적 타입을 결정하는 프로그래밍 기법

## 사용하는 이유
- 입력에 대하여 범용적인 클래스나 함수를 만들 때 반환 값의 타입을 확신 할 수 없다
    - 제네릭은 가능하다.
```
class Stack {
    private data: any()
}
```
