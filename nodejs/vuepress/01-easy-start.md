# Start VuePress
Vuepress로 Static Site만들기

## fast start!

yarn으로 **vuepress** 설치
```
$ yarn global add vuepress #npm install -g vuepress
```

README.md 생성
```
$ echo '# Hello VuePress' > README.md

```
http://localhost:8080 dev모드 시작
```
$ vuepress dev
```

vuepress 빌드
```
$ vuepress build
```
